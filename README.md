#  GlucoSense

An AI-powered system designed to predict diabetes using Machine Learning techniques.

###  Key Highlights

*   Smart Prediction: An advanced Machine Learning project that predicts diabetes risk with high accuracy.
*   Predictive Modeling: Built using Random Forest and Logistic Regression for robust results.
*   Data Insights: Features an in-depth analysis of health indicators like Glucose, BMI, and Age.
*   Smart Alert System: Generates medical-grade alerts based on risk probability.
*   Visual Analysis: Includes Heatmaps and Feature Importance charts.

###  Tech Stack
Python | Google Colab | Scikit-learn | Seaborn | Pandas
