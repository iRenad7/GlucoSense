# GlucoSense
An AI-powered system designed to predict diabetes using Machine Learning techniques.
GlucoSense: Smart Diabetes Prediction
An advanced Machine Learning project that predicts diabetes risk with high accuracy.
Key Highlights:
• Predictive Modeling: Built using Random Forest and Logistic Regression for robust results.
• Data Insights: Features an in-depth analysis of health indicators like Glucose, BMI, and Age.
• Smart Alert System: Generates medical-grade alerts based on risk probability (High, Medium, or Low Risk).
• Visual Analysis: Includes Heatmaps and Feature Importance charts to visualize data correlations.
💻 Tech Stack:
Python | Google Colab | Scikit-learn | Seaborn | Pandas.
